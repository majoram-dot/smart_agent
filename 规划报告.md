# SmartAgent 项目规划报告

## 📋 项目概述

**项目名称**: SmartAgent  
**项目类型**: AI Agent 框架（源码整合重构）  
**目标**: 整合多个领先开源项目的核心功能，构建一个独立、全功能的通用 AI Agent 框架。
**最后更新**: 2026-03-05（第二阶段重构完成后）

## 🔍 源码来源分析

### 1. 核心 Agent 基座 (HuggingFace 风格)
- **定位**: 轻量级生产级 Agent 框架
- **核心贡献**:
  - `CodeAgent` / `ToolCallingAgent` — Agent 核心
  - `LocalPythonExecutor` — 本地代码执行
  - `AgentMemory` — 运行时步骤记忆
  - `LiteLLMModel` — 支持 100+ LLM
  - 搜索工具 (DuckDuckGo / Google / Wiki)
  - `RemotePythonExecutor` — 远程执行 (E2B / Docker / Modal / Blaxel)
  - `GradioUI` — Web 界面

### 2. 工具集成与协议转换层 (Legacy Tools)
- **定位**: Agent 教程与示例集合
- **核心贡献**:
  - `MemoryTool` — 4 种长期记忆（工作 / 情景 / 语义 / 感知）
  - `MemoryManager` — 记忆管理系统
  - `NoteTool` — 笔记工具
  - `TerminalTool` — 终端工具
  - `MCPTool` — MCP 工具连接
  - RAG 高级检索（MQE / HyDE / 混合检索）
  - GSSC Pipeline — 生成-存储-搜索-组合流水线
  - MarkItDown Pipeline — 文档转换

### 3. 轻量化向量索引与检索引擎 (Tiny RAG Module)
- **定位**: 轻量级 RAG 实现
- **核心贡献**:
  - `VectorStore` — 向量存储（HNSW 索引）
  - 7 种 Embedding 模型
  - `BgeReranker` — 重排序
  - `ReadFiles` / `Documents` — 文档加载
  - 多 LLM 封装（OpenAI / DashScope / Zhipu / Siliconflow）

## 🏗️ 分层架构（重构后）

重构核心原则：**单向依赖、从底向上**，各层仅允许依赖其下方的层，禁止反向引用。

```
base  →  tools / memory / monitoring / executors  →  agents  →  deep_research / ui
```

| 层级 | 模块 | 说明 |
|------|------|------|
| **base**（最底层） | `base/tool.py` | 无任何业务依赖，所有工具的抽象根 |
| **服务层** | `tools`, `memory`, `monitoring`, `executors`, `llm`, `rag`, `serialization` | 各自独立，通过相对路径互不干扰 |
| **协调层** | `agents/core.py` | 依赖所有服务层，为唯一具有横向整合权的模块 |
| **功能扩展层** | `deep_research`, `evaluation` | 构建在协调层之上的高层功能 |
| **接口层** | `ui`, `cli` | 最外层，只做 I/O 代理与展示 |

## ⚙️ 技术栈

### 核心依赖
- Python >= 3.10
- `litellm >= 1.60.2`（LLM 统一接口）
- `python-dotenv`（环境变量管理）
- `rich`（终端美化输出）
- `jinja2`（提示词模板）
- `hnswlib` + `numpy`（向量索引）
- `huggingface_hub`（模型/工具上下传）
- `duckduckgo-search`（WebSearchTool）

### 可选依赖
- `gradio`（Web UI）
- `e2b-code-interpreter` / `docker`（远程执行）
- `sentence-transformers`（本地 Embedding）
- `faiss-cpu`（向量检索备选）
- `rank-bm25`（混合检索）

### 评估专有依赖（独立测试环境）
- `bfcl-eval`（工具调用能力评测）
- `datasets`（GAIA 基准集下载）

## 🔧 环境配置

### 推荐安装方式

```bash
conda create -n smart_agent python=3.11 -y
conda activate smart_agent
pip install -e .                   # 可编辑模式安装，直接读取 src/
```

### API 配置（`.env`）

```bash
# 阿里云 DashScope（已验证）
OPENAI_API_KEY=<your-key>
OPENAI_BASE_URL=https://dashscope.aliyuncs.com/compatible-mode/v1
MODEL_NAME=qwen-plus

# 搜索工具（可选）
TAVILY_API_KEY=<your-key>
SERPAPI_API_KEY=<your-key>

# GAIA 评估（可选）
HF_TOKEN=<your-huggingface-token>
```

### 评估专属独立环境

```bash
conda create -n smart_agent_eval python=3.11 -y
conda activate smart_agent_eval
pip install -e .
pip install -r benchmarks/requirements-eval.txt
```
